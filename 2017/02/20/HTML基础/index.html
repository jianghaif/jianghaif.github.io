HTML基础笔记


HTML基础的标签记录和使用

##HTML基础

: 段落标签

标签中可以添加属性

属性在开始的标签中 与标签相隔一个空格属性名称应被引号保温并接上等号

嵌套

将一个标签置于其他标签之中——这被称作嵌套

</p>段落中的<stong>内容</stong></p>
strong标签表示强调p中其他不变但是“内容”被strong修饰了

空元素

有些元素不包含内容。被称之为空元素

<img src="图片" alt="显示文本"> 
特别alt:规定图片无法显示时的代替显示的文本内容

HTML所必需的元素

1. <html></html>:包含了整个页面的内容，有时也被称作根元素。
2. <head></head>:包含了所有和页面相关的信息，但是不会被用户所看到
3. <body></body>:包含了你想被用户看到的内容
4. <meta charset="utf-8">:指定了文档需要使用的字符编码
5. <title></title>:设置了页面的标题，标题显示在浏览器标签页上，
 而且在将网页添加到收藏夹或喜爱中时将作为默认名称

  1. <html>
   <head>
      <meta charset="utf-8">
      <title>标题</title>
   </head>
   <body>
    <img src="图片" alt="无法显示">
    <p>一段实例</p>
    </body>
</html>

图像image

<img src="图片路径" alt="图片描述">

将图像嵌入到元素所在位置。通过图像文件路径 src属性实现这一功能。

但是这一元素需要包括alt属性 对图片无法显示时进行图片文字描述

##标记文本

标题

HTML 包括六个级别的标题，，虽然可能只会用到 3-4 最多。

<h1>–</h1>
    ...
<h6>-</h6>

段落

<p>内容</p>

<p> 元素是用来指定段落的。可以用它来指定常规的文本内容

 空格

将&nbsp;插入html标签中显示空格

列表

Web上的内容都是列表，要标记列表通常包括至少两个元素最，

常用的列表类型是有序列表（ ordered lists）和无序列表（ unordered lists）

  1. 无序列表 中项目的顺序并不重要。这些内容被包括在一个 ul 元素里。
  2. 有序列表 中项目的顺序很重要，就像一个食谱。这些内容被包括在一个 ol 元素里。

列表内的每个项目被包括在一个 li (list item)元素里

<p>At Mozilla, we’re a global community of</p>
    /*将p标签中的元素改成列表*/
<ul> 
  <li>technologists</li>
  <li>thinkers</li>
  <li>builders</li>
</ul>
<ol> 
  <li>technologists</li>
  <li>thinkers</li>
  <li>builders</li>
</ol>

链接

要植入一个链接，我们需要使用一个简单的link — \ — a 是 “anchor” （锚）的缩写。要将一些文本添加到链接中，只需如下几步：

添加一些文本。我们选择“Mozilla Manifesto”。

将文本包含在 元素内，就像这样

<a>Mozilla Manifesto</a>

赋予 元素一个 href 属性，就像这样：

<a href=""></a>

把想要链接的网址放到 href 属性内：

<a href="https://www.mozilla.org/en-US/about/manifesto/">Mozilla Manifesto</a>

表头由th标签定义
